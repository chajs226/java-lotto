# 로또
## 기능 목록
* 구입 금액을 입력한다.
  * 로또 1장의 가격은 1000원
* 로또 번호를 생성한다.
  * 1~45사이의 랜덤한 숫자 6개 생성
  * 로또 1장 내 중복된 번호를 가지지 않는다.
  * 로또 1장 내 번호는 오름차순 정렬된다.
* 입력된 구입 금액으로 구매 가능한 갯수를 구한다. 
* 구매 가능한 갯수 만큼 로또를 구매한다.
* 지난주 당첨 번호를 입력한다.
* 보너스 볼을 입력한다.
* 구매한 로또와 당첨 번호를 비교한다.
  * 1개 일치
* 당첨 결과, 통계를 보여준다.
  * 3개 일치 (5000원) - 0개
  * 4개 일치 (50000원) - 0개
  * 5개 일치 (1500000원) - 0개
  * 5개 일치, 보너스 볼 일치 (30000000원) - 0개
  * 6개 일치 (2000000000원) - 0개
  * 수익률 표시 = 당첨금/구매금액
